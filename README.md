# Анализ детского сна: данные за 5 лет

Анализ данных о сне двух детей (Василиса 5 лет и Забава 2 года) за 3 года наблюдений.

## Файлы в проекте

- vasilisa_sleep.xlsx, zabava_sleep.xlsx - исходные данные Excel
- sleep_analysis.ipynb - весь анализ в Jupyter Notebook
- Разные .csv файлы - обработанные данные

## Как запустить

Установите зависимости:
pip install pandas numpy matplotlib seaborn jupyter openpyxl

Запустите ноутбук:
jupyter notebook sleep_analysis.ipynb

## Что анализируем

1. Базовые метрики сна:
   - Сколько в среднем спят
   - Сколько снов в сутки
   - Когда ложатся и встают

2. Интересные графики:
   - Распределение сна (3 пика: короткий, дневной, ночной)
   - Как меняется сон с возрастом
   - Сравнение детей в одинаковом возрасте
   - Тепловая карта сна по времени

3. Полезные выводы:
   - Оба ребенка в пределах норм ВОЗ
   - Режим стабильный: ночь 21-22, день 13-15
   - С возрастом снов меньше, но они длиннее
   - Василиса спит по нижней границе нормы - это ее особенность
